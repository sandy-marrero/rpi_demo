## Running Instructions

1. **Install Dependencies**  
   Ensure the following are installed on both Raspberry Pis:
   - Python 3
   - Flask
   - RPi.GPIO
   - requests
   - sqlite3
   - python-dotenv

2. **Hardware Setup**  
   Connect an LED to GPIO pin 26 with a resistor.

3. **Run the Script**  
   Run the script on both Raspberry Pis using the command:
   ```
   python3 demo.py
   ```
